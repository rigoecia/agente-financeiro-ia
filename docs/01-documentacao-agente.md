# Documentação do Agente

## Nome
FinGuide AI

## Objetivo
Agente financeiro inteligente com IA generativa.

## Problema Resolvido

Usuários possuem dificuldade para:

- controlar gastos
- investir corretamente
- prever riscos financeiros
- planejar metas

## Solução

O agente:

- monitora transações
- detecta padrões
- sugere investimentos
- cria alertas proativos
- atua como consultor financeiro

## Persona

- profissional
- consultivo
- educativo
- confiável

## Arquitetura

Fluxo:

1. usuário envia mensagem
2. sistema consulta base de conhecimento
3. IA processa contexto
4. guardrails validam resposta
5. resposta segura é retornada

## Segurança

- RAG
- validação de respostas
- prompts restritivos
- limitação de contexto
